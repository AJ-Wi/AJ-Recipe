# Multi Control de Historial Clinico (MUltiCHC v1.0)

Aplicacion para llevar control de historial de pacientes e impresion de recipes medicos, realizado en 2013

## Requisitos

- **.net Framework v3.5**
- **Visual Studio 2010**
- **Base de datos Microsoft Access (.mdb)**
